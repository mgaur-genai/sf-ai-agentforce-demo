# Salesforce AI & Agentforce Demo Repository

## Overview
This repository contains a collection of hands-on projects focused on building AI-powered, agentic workflows within the Salesforce ecosystem.

The goal of this repository is to demonstrate practical implementation of:
- AI automation patterns
- Agentforce-based agents and actions
- External AI orchestration with Salesforce
- MCP (Model Context Protocol) integrations
- Real-world CRM use cases enhanced with AI

This serves as a portfolio of applied Salesforce AI architecture and development.


## Key Concepts Covered

### AI Automation
- Structured prompt engineering
- JSON-based deterministic outputs
- Tool/function calling patterns
- AI-driven decision making
- Logging, validation, and governance

### Agentforce (Salesforce AI Agents)
- Agent configuration and orchestration
- Custom actions and integrations
- Salesforce-native AI workflows
- Context-aware automation

### MCP (Model Context Protocol)
- Tool and resource abstraction for AI systems
- External system integration patterns
- Standardized AI-to-tool communication

### Salesforce Platform
- Experience Cloud (public portals)
- Apex (integration layer)
- Lightning Web Components (LWC)
- Flow-based automation
- Data modeling and governance
